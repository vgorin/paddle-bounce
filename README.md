# Paddle Bounce

**Paddle Bounce** is a minimalist arcade game built around a single goal: keep the ball in play using a responsive paddle. The game focuses on timing, precision, and maintaining control as the ball speeds up.

---

## Gameplay

- Control a paddle to keep the ball from falling.
- The ball accelerates over time, increasing the challenge.
- The objective is to survive as long as possible.

---

## Controls

- Move the paddle horizontally using the mouse.
- Keep the ball in play to achieve a higher score.

---

## Running the Game

Open `index.html` in any modern browser.  
No installation or dependencies required.
